# Проект: Автоматическое заполнение ФИО преподавателей в нагрузке

## Что делает программа?

Программа автоматически заполняет столбец **"ФИО"** в новом файле Excel с учебной нагрузкой. Она берет данные из файла прошлого года и для каждой пары **"Дисциплина + Вид нагрузки"** находит преподавателя, который вёл эту дисциплину в прошлом году.

**Важное условие:** ФИО копируется только если в прошлом году по данной дисциплине и виду нагрузки был **один преподаватель**.

---

## Какие файлы нужны?

### Исходные файлы:
1. **`15_вариант.xls`** — данные прошлого года (формат `.xls`)
2. **`нагрузка-вар2-ТиСАПРМП-2023-2024.xlsx`** — новый файл с нагрузкой (формат `.xlsx`)

### Результат:
- **`задача5_результат.xlsx`** — новый файл с заполненными ФИО

---

## Как работает программа?

### 1. Загрузка данных прошлого года
Программа читает файл `15_вариант.xls` и превращает каждую строку в словарь, где:
- Ключ = название столбца
- Значение = данные из ячейки

### 2. Анализ данных прошлого года
Создается словарь, где:
- **Ключ** = пара `(Дисциплина, Вид нагрузки)`
- **Значение** = множество ФИО преподавателей

Пример:
```python
teacher_key = {
    ("Математика", "Лекции"): {"Иванов И.И."},  #один преподаватель
    ("Физика", "Практика"): {"Петров П.П.", "Сидоров С.С."}  #два преподавателя
}
```

### 3. Заполнение нового файла
Программа проходит по всем строкам нового файла:
- Для каждой строки берет **Дисциплину** и **Вид нагрузки**
- Ищет такую же пару в данных прошлого года
- Если находит и там **ровно один преподаватель** — копирует ФИО

---

## Требования к данным

Файлы должны содержать столбцы с названиями:
- `"Дисциплина"`
- `"Вид нагрузки"`
- `"ФИО"`

Программа автоматически найдет их по заголовкам.

## Как устроен код?

### Библиотеки:
- `xlrd` — для чтения старых файлов `.xls`
- `openpyxl` — для работы с новыми файлами `.xlsx`

### Основные функции:

#### `download_ly(file_path)`
Читает Excel-файл и возвращает список словарей.

**Как работает:**
1. Открывает книгу Excel
2. Берет первый лист
3. Читает заголовки из первой строки
4. Для каждой следующей строки создает словарь

#### `task_5(lines_ly, ws_new, quant_disc, quant_type, quant_name)`
Основная логика — копирует ФИО в новый файл.

**Как работает:**
1. Анализирует данные прошлого года
2. Строит словарь преподавателей по парам
3. Проходит по строкам нового файла
4. Если в прошлом году был 1 преподаватель — копирует ФИО

---

## Важные моменты

1. **Программа копирует ФИО только если был ровно ОДИН преподаватель** — это главное условие

2. **Файл прошлого года должен быть в формате `.xls`** — для этого используется библиотека `xlrd`

3. **Новый файл должен быть в формате `.xlsx`** — для этого используется библиотека `openpyxl`

4. **В новом файле должен быть лист с названием "Нагрузка"**

5. **Все названия столбцов должны совпадать** с теми, что указаны в программе

---

## Возможные проблемы

### Ошибка: "не найден столбец"
**Решение:** Проверьте, что в новом файле есть столбцы:
- `"Дисциплина"`
- `"Вид нагрузки"`
- `"ФИО"`

### Ошибка: "File not found"
**Решение:** Проверьте, что файлы находятся в той же папке, что и программа

### Пустые ФИО
**Решение:** Возможно, в прошлом году по этой паре было несколько преподавателей, или данных нет
